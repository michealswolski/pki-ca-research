# PKI & Certificate Authority Vulnerability Research
> Cybersecurity Coursework — Eastern Michigan University

## Overview
Research project analyzing Public Key Infrastructure vulnerabilities, certificate authority trust models, and real-world CA compromise case studies. Produced as an academic deliverable and internship work sample.

## Research Areas
- **PKI Architecture**: How certificate chains, root CAs, and intermediate CAs establish trust
- **CA Vulnerabilities**: Common attack vectors against certificate authorities
- **Case Studies**: Analysis of notable real-world CA compromises and their impact
- **Trust Model Analysis**: Strengths and weaknesses of the current web PKI model
- **Mitigation Strategies**: Certificate Transparency, HPKP, CAA records, and modern defenses

## Tools & Frameworks Referenced
| Tool / Standard | Context |
|-----------------|---------|
| X.509 Certificates | Certificate format and structure analysis |
| Certificate Transparency | Log-based CA accountability mechanism |
| OCSP / CRL | Certificate revocation checking methods |
| Let's Encrypt / ACME | Automated CA issuance model |

## Deliverable
Comprehensive research report submitted as academic coursework and used as a professional writing sample.

## What I Learned
- Deep understanding of PKI trust chains and their failure modes
- How certificate revocation works (and where it breaks down)
- Real-world impact analysis of CA compromises on internet security
- Technical writing for cybersecurity research audiences
